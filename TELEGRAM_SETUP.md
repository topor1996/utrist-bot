# Создание бота в Telegram

## Шаг 1: Создание бота через @BotFather

1. **Откройте Telegram** и найдите бота **@BotFather**
   - Это официальный бот для создания ботов
   - Иконка: синий робот с гаечным ключом

2. **Отправьте команду** `/newbot`
   - Или нажмите кнопку "New Bot" в меню

3. **Введите имя бота** (отображается в списке контактов):
   ```
   Ваш юрист СПб
   ```
   - Можно использовать любое имя
   - Это имя видят пользователи

4. **Введите username бота** (должен заканчиваться на `bot`):
   ```
   vash_urist_spb_bot
   ```
   - Должен быть уникальным
   - Если занят, попробуйте другой вариант
   - Примеры: `urist_spb_bot`, `vash_urist_bot`, `urist_consult_bot`

5. **Скопируйте токен**, который даст @BotFather:
   ```
   123456789:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
   ```
   - **ВАЖНО:** Сохраните этот токен!
   - Он понадобится для настройки на Railway
   - Не показывайте его никому

## Шаг 2: Получение вашего Telegram ID

1. Найдите бота **@userinfobot** в Telegram
2. Отправьте команду `/start`
3. Бот покажет ваш ID (число, например: `123456789`)
4. **Скопируйте этот ID** - он нужен для админ-панели

## Шаг 3: Дополнительные настройки (опционально)

После создания бота можно настроить:

### Описание бота:
```
/setdescription
```
Выберите вашего бота и введите описание:
```
Юридические услуги в Санкт-Петербурге. Консультации, регистрация ИП/ООО, судебное сопровождение.
```

### Текст "О боте":
```
/setabouttext
```
Выберите бота и введите:
```
Юридическая компания "Ваш юрист" в Санкт-Петербурге. Запись на консультацию, вопросы юристу.
```

### Команды бота:
```
/setcommands
```
Выберите бота и введите список команд:
```
start - Начать работу с ботом
admin - Админ-панель (только для администраторов)
```

### Фото бота:
```
/setuserpic
```
Выберите бота и отправьте фото

## Шаг 4: Проверка

1. Найдите вашего бота в Telegram (по username, который вы указали)
2. Отправьте команду `/start`
3. Бот должен ответить приветственным сообщением

## Что дальше?

После создания бота у вас будет:
- ✅ **BOT_TOKEN** - токен бота (от @BotFather)
- ✅ **ADMIN_IDS** - ваш Telegram ID (от @userinfobot)

Эти данные нужно будет добавить в Railway в разделе "Variables":
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id
```

## Важно:

- Токен бота - это как пароль, не показывайте его никому
- Если токен утерян, можно получить новый через @BotFather командой `/token`
- Username бота нельзя изменить после создания

## Проблемы?

- **Бот не отвечает:** Убедитесь, что бот запущен на Railway
- **Ошибка токена:** Проверьте, что токен скопирован полностью
- **Бот не найден:** Проверьте username, он должен быть уникальным
